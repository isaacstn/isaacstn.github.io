一、是非題 （每題 2 分）
閱讀以下題目之敘述，若認為正確請寫 'T'，錯誤請寫 'F'，不知道則可以寫 'N'。

  1. 對任何推論來說，只要前提都正確，而且推論也合理，那結論就一定會是正確的。

    2. F。不，對演繹推論而言是如此，但對非演繹推論而言，即使前提正確且推論也合理，仍不能保證結論一定正確。

  2. 一個推論是否合理，取決於推論的前提能否支持結論，若前提支持結論則推論就是合理的。

    3. T。是，推論合理即意謂著前提可以支持結論，只是演繹推論和非演繹推論中的「前提支持結論」之關係的理解略有差異。

  3. 所有的推論都包含三個部分：推論的前提、推論的假設、推論的結論。

    4. F。不，所有推論都只有兩個部分：前提、結論。所謂「推論的假設」其實就是指前提而言。

  4. 非演繹推論的特色就是其前提往往不能百分之百地保證結論一定為真。

    5. T。是的，非演繹推論和演繹推論間的主要差異便在於，對一個合理的非演繹推論而言，其前提也不見得能保證結論正確。

  5. 推論的前提是否支持結論的判準是唯一的：前提必須能保證結論一定成立才算是支持結論。

    6. F。並非如此，非演繹推論的「前提支持結論」之關係就不是如此。

  6. 推論通常是在無法直接確認問題答案時才會採取的一種解答問題的手段。

    7. T。是，若可直接確認問題的答案通常就不需要做推論了。

  7. 古人發現太陽每天都東升西落，因此判斷太陽只會東升西落。這是在做歸納推論。

    8. T。用重複的經驗來推論一個規律性就是歸納推論。

  8. 獵人根據足跡的大小、形狀、深淺等特徵來追蹤野獸的去向。這是在做歸納推論。

    9. F。用野獸行走時留下的結果推論原因 (野獸的去向) 是在做溯因推論而非歸納推論。

  9. 維修人員根據經驗判斷電腦之所以不能開機是由於記憶體損毀。這是溯因推論的應用。

    10. T。經驗提供零件故障與不能開機之間的因果關係，因此是由結果推論原因的溯因推論沒錯。

  10. 小明發現每次喝完咖啡晚上都睡不好，故認定咖啡就是妨礙睡眠的原因。這是溯因推論的應用。

    11. F。由重複的經驗 (喝咖啡和睡不著) 去推論出兩者間的規律 (喝咖啡導致睡不著)，因此是在做歸納推論而非溯因推論。

  11. 在做溯因推論時，必須先對與事件相關的因果關係有所瞭解，否則便無法進行推論。

    12. T。如果不先瞭解與事件相關的因果關係，就算調查出了結果也無法判斷有哪些原因可能導致這些結果，因此也就無法進行溯因推論。

  12. 在做歸納推論時，如果不能取得超過半數案例的資料，就絕不能將結論推廣到整個群體。

    13. F。不見得，例如藥物測試就不可能取得超過半數人口的資料。重點在於抽樣調查的樣本是否對母群體有足夠的代表性。

  13. 「塑膠花是假花」、「男人的情話都是假的」，兩個語句中的「假」字意思相同。

    14. F。不相同，前者是區分正品和贗品，而後者則是區分語句的真假。

  14. 「草船借箭的故事是假的」、「大腳哈利的傳聞也是假的」，兩句話中的「假」字意思相同。

    15. T。是，兩者都是在區分一件事情是否真實發生過，還是被虛構出來的。

  15. 「演員在台上的表演是假的」、「劇中使用的道具也是假的」，兩句話中的「假」字意思相同。

    16. T。是，兩者都是在一個事物是模仿的，演員的表演和劇中的道具都是在模仿現實。

二、簡答題 (1) 判斷題目中的兩個語句間的蘊涵關係（每題 2 分）
註1．對任意兩語句$X$和$Y$而言，$X$蘊涵$Y$ $=_{df}$ 只要$X$為真則$Y$就一定為真。
註2．任意兩語句$X$和$Y$之間的蘊涵關係共有四種可能：$X$蘊涵$Y$、$Y$蘊涵$X$、互相蘊涵、互不蘊涵。

  1. A：所有人都會死。
    B：沒有人不會死。
    2. 互相蘊涵。
    只要所有人都會死，就沒有人不會死；只要沒有人不會死，則所有人就都會死。
  2. A：早起的鳥有蟲吃。
    B：晚起的鳥沒蟲吃。
    3. 互不蘊涵。
    早起的鳥有蟲吃不代表晚起的鳥就沒蟲吃，可能隨時都有充足的蟲；反之，晚起的鳥沒蟲吃也不表示早起的鳥就有蟲吃，可能正在鬧蟲荒，誰都沒有蟲可以吃。
  3. A：我的身高比你高。
    B：我的身高不比你矮。
    4. A蘊涵B。
    我的身高比你高，那就一定不比你矮；但我的身高若只是不比你矮，也可能只跟你一樣高，不見得就會比你高。
  4. A：今天小明上課遲到。
    B：今天有人上課遲到。
    5. A蘊涵B。
    小明今天上課遲到，那就一定有人上課遲到（就是小明）；但若有人上課遲到，則不見得就是小明遲到（也可能是別人）。
  5. A：我只吃我自己做的食物。
    B：我只做我自己吃的食物。
    6. 互不蘊涵。
    我只吃自己做的食物，表示別人做的食物我不吃，但我可以做食物給別人吃；我只做自己吃的食物，表示我不會做食物給別人吃，但別人做的食物我可以吃。
  1. A：整個八月裡每天都在下雨。
    B：整個八月裡沒有一天不下雨。
    2. 互相蘊涵。
    只要每天都在下雨，就沒有一天不下雨；只要沒有一天不下雨，就表示每天都在下雨。
  2. A：人必須持續運動才能維持好身材。
    B：人只要持續運動就能維持好身材。
    3. 互不蘊涵。
    必須運動才能維持好身材，意思是不運動不行，但運動了也不一定就能維持好身材；只要運動就能維持好身材，意思是運動一定能維持好身材，但也不是非運動不可。
  3. A：人的外在美不比內在美更重要。
    B：人的內在美比外在美更重要。
    4. B蘊涵A。
    外在美不比內在美重要，不代內在美就更重要，也可能一樣重要；但內在美若比外在美重要，則外在美一定不比內在美重要。
  4. A：有些哺乳類動物是胎生動物。
    B：大部分的哺乳類動物都是胎生動物。
    5. B蘊涵A。
    有些哺乳類是胎生不表示大部分都是，也可能只有少部分是；但若大部分哺乳類是胎生，那一定有些是。
  5. A：楊過已經拜了小龍女為師父。
    B：小龍女已經收了楊過為徒弟。
    6. 互相蘊涵。
    楊過既已拜了小龍女為師父，就表示小龍女已接受楊過做其徒弟；而小龍女若已收了楊過為徒，就表示楊過一定也已接受小龍女做其師父。

三、簡答題 (2) 判斷題目中的兩個概念間的蘊涵關係（每題 2 分）
註1．對任意兩概念$X$和$Y$而言，$X$蘊涵$Y$ $=_{df}$ 只要「$a$是$X$」為真則「$a$是$Y$」就一定為真。
註2．任意兩概念$X$和$Y$之間的蘊涵關係共有四種可能：$X$蘊涵$Y$、$Y$蘊涵$X$、互相蘊涵、互不蘊涵。

  1. A：「滑鼠」；B：「老鼠」
    2. 互不蘊涵。
    滑鼠不是老鼠；老鼠也不是滑鼠。
  2. A：「正三角形」；B：「等邊三角形」
    3. 互相蘊涵。
    正三角形的三邊相等，故是等邊三角形；等邊三角形既然三邊相等，那就是正三角形。
  3. A：「襪子」；B：「過膝襪」
    4. B蘊涵A。
    襪子不見得都過膝；但過膝襪一定是襪子。
  4. A：「鞋子」；B：「長靴」
    5. B蘊涵A。
    鞋子不見得都是長靴；但長靴一定是鞋子。
  5. A：「洋芋」；B：「馬鈴薯」
    6. 互相蘊涵。
    洋芋就是馬鈴薯；馬鈴薯就是洋芋。
  1. A：「火雞」；B：「雞」
    2. 互不蘊涵。
    火雞不是雞；雞也不是火雞。
  2. A：「陌生人」；B：「不認識的人」
    3. 互相蘊涵。
    陌生人一定都是不認識的人；而且不認識的人也都是陌生人。
  3. A：「動物」；B：「生物」
    4. A蘊涵B。
    動物都是生物；但生物不見得是動物。
  4. A：「杯子」；B：「容器」
    5. A蘊涵B。
    杯子都是容器；但容器不一定是杯子。
  5. A：「魚」；B：「鯨魚」
    6. 互不蘊涵。
    魚不是鯨魚；鯨魚也不是魚 (是哺乳類)。

四、問答題 判斷以下推論是不是有效推論，並說明理由。（每題 5 分）
註1．一個推論是有效推論 $=_{df}$ 只要推論的前提皆為真則結論就一定也為真。
註2．判斷方式：先假設前提皆為真，再判斷於前提皆真的假設下結論是否一定為真。

  1. 是人就會犯錯。
  2. 黃頌竹是人。
  3. 黃頌竹會犯錯。
    4. 答：有效。因為當前提皆真時，表示所有人都會犯錯，而且黃頌竹也在所有人的範圍內，因此他當然會犯錯。故只要前提皆為真，則結論也一定為真。
  1. 塑膠不會導電
  2. 教室裡的花瓶不會導電。
  3. 教室裡的花瓶是塑膠做的。
    4. 答：無效。即使塑膠不導電，也不見得是唯一不導電的材料，故教室裡不導電的花瓶也可能是用塑膠以外的不導電材料所製。因此就算前提皆真，結論也不見得真。
  1. 小明去過日本或韓國。
  2. 小明去過日本。
  3. 小明沒去過韓國。

    答：無效。
    「或」的意思是「至少一個成立」，故前提1表示他至少去過其中一國，因此也可能兩國都去過。所以就算他去過一國也不表示沒去過另一國。

  1. 只要下雨地面就會積水。
  2. 只要地面積水就會增加車禍風險。
  3. 只要下雨就會增加車禍風險。

    答：有效。
    既然有下雨就一定會令地面積水，而地面積水就一定會令車禍風險增加，那麼下雨當然就一定會令車禍的風險增加。

  1. 犯人是司機或園丁。
  2. 如果犯人是司機，那兇手就是小明。
  3. 如果犯人是園丁，那兇手就是小華。
  4. 兇手是小明或小華。

    答：有效。
    前提2說犯人是司機的話兇手就是小明，前提3又說犯人是園丁的話兇手就是小華，那麼在犯人只能是司機或園丁的情況下，兇手自然只能是小明或小華。

  1. 他只要睡過頭上學就會遲到。
  2. 他只要上學遲到就會被老師罰站。
  3. 他今天沒有睡過頭。
  4. 他今天沒有被老師罰站。

    答：無效。
    就算他今天沒睡過頭，也不表示他就不會因其他緣故而遲到；就算他今天沒有遲到，也不表示他就不會因其他緣故而被老師罰站。

  1. 有些貓跑得很快。
  2. 有些貓跳得很高。
  3. 有些貓跑得很快也跳得很高。

    答：無效。
    前提1說有些貓跑得快，前提2說有些貓跳得高，但前提並未表示兩者間一定有重複的，所以根據前提無從得知是否有些貓跑得快又跳得高。

  1. 所有鳥都會飛。
  2. 我不會飛。
  3. 我不是鳥。

    答：有效。
    既然前提1假定了所有的鳥都會飛，那就沒有不會飛的鳥，因此當前提2又假定我不會飛，我便不可能是一隻鳥，否則就會有不會飛的鳥了。
