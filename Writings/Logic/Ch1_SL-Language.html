語句邏輯的語言

我在前一章裡提到，我們的語言有一個重要的特性是組合性，說一個語言是組合性的，意思是說，這個語言中有一些能夠承載意義的最基本元素，此外，在這個語言中所有能夠承載意義的表達式，一定都是用這些基本元素所構成的，而且這些表達式的意義都是被那些基本元素的意義所決定的。

在討論到我們日常生活中使用的中文這個語言時，我說中文裡最基本的元素雖然是個別的文字，但是能夠承載意義的最基本元素其實是語詞，這是因為中文裡其實有部分文字單獨來看是沒有意義的，比如說「葡」和「萄」只有當連在一起寫成「葡萄」時才有意義，這兩個字個別來看卻是沒有意義的。

最後我指出，一個組合性的語言可以只包含有限數量的基本元素（在中文裡就是語詞），卻可以創造出無限多個表達不同意義的表達式（語句），如此一來既能大幅降低溝通成本，也能大幅增加語言的表達效能。而邏輯學家為了研究推論而發明的語句邏輯系統，當然也具備了組合性這個特色。

語句邏輯的語言元素

語句邏輯 (sentential logic) 的語言一樣具有組合性，也就是說，語句邏輯的語言一樣有能夠承載意義的最基本元素，而且語句邏輯中的所有能夠表達意義的表達式，無論多複雜，都是由這些基本元素所構成的。不過和中文不一樣的地方是，語句邏輯的語言中能夠承載意義的最基本元素是語句，而不是語詞。舉例來說：

  1. 海綿寶寶住在深海的大鳳梨裡。
在中文裡，我們可以將這個語句大致區分成「海綿寶寶」、「住」、「在」、「深海的」、「大」、「鳳梨」、「裡」等七個語詞，不過在語句邏輯中，這個語句就只會被當成是一個基本的整體，不能再被分割，因為它就是語句邏輯的語言中能夠承載意義的最基本元素。

為了便於記載，我們在語句邏輯的語言中，一律以大寫的英文字母來代表個別的語句，如：$'A'$、$'B'$、$'C'$……等。雖然大寫的英文字母總共只有26個，而能夠在自然語言中表達的語句卻有無限多個，因此是遠遠不夠的。但在大部分的時候，我們要處理的論證多半不會用到太多的語句，所以26個字母也還算夠用。

但考慮到可能在某些特殊的情況下，我們會在一個論證裡就使用到超過26個語句，因此我們也可以接受在大寫的英文字母上加上自然數下標，來增加可用的語句代號，如：$'A_1'$、$'B_3'$、$'C_13'$……等。由於自然數有無限多個，我們在語句邏輯的語言中就會有無限多個語句代號可以使用了。

當然，如果在語句邏輯的語言中只有用英文字母來代表的語句，那麼其實做不了什麼推論，也無法發揮語言的組合性的功用。因此，語句邏輯的語言中還需要一些元素讓這些語句之間可以組合起來，形成更複雜的語句，而這些元素就被稱為「語句連接詞」，或者也可以簡單地稱為「連接詞」(connectives) 即可。

在我們這堂課上會用到的語句邏輯系統中，一共有五個語句連接詞，分別是：$'¬'$、$'∧'$、$'∨'$、$'→'$、$'↔'$，如果要對應到中文裡的語詞的話，可以將上述五個連接詞的前面四個分別理解為「並非……」、「……而且……」、「……或者……」、「如果……，則……」，至於最後一個連接詞比較特別，它在中文裡沒有直接對應的語詞，學界一般將它翻譯成「……當且僅當……」，英文寫作 "... if and only if..."。

除了最後一個語句連接詞以外，相信前面四個語句連接詞在日常語言的中文裡都很常見，意思也很好理解。「並非……」就是用來否定某個句子；「……而且……」則是用來肯定兩個句子都是真的；而「……或者……」則是用來表達兩個句子中至少有一個是真的這裡要注意的是，中文裡的「或者」有時候是用來表達「至少有一個是真的」，但有時候也會用來表達「至少有一個是真的，而且只有一個是真的」。不過在語句邏輯中的「……或者……」只有前一種意思，因此它可以允許連接的兩個句子都為真的情況。；至於「如果……，則……」就是用來表達一個條件句，而條件句則是用來表達被連接的兩個句子之間的蘊涵關係。

蘊涵關係是一個單向的關係，當我們將兩個句子以「如果……，則……」這個連接詞連接起來時，寫在「如果……」那一側的句子被稱為條件句的前件，寫在「則……」這一側的被稱為條件句的後件。當我說一個條件句表達的是兩個句子之間的蘊涵關係時，意思就是條件句的前件蘊涵條件句的後件。而當條件句的前件蘊涵條件句的後件時，就代表只要前件為真，那麼後件就會為真。此外，當條件句的前件蘊涵條件句的後件時，我們也可以說前件是後件的充分條件。有注意到了嗎？這個說法和有效論證的定義其實一模一樣。因此，當一個論證是有效論證時，我們也可以說這個論證的前提蘊涵結論，或說這個論證的前提是結論的充分條件。

最後，當我們以「……當且僅當……」這個連接詞將兩個句子連接起來時，其實是在表達兩個條件句的合併，因此也被稱為雙條件句。為了做出區別，所以我們就將前面提到的條件句稱為單條件句。前面說過，單條件句表達的意思是「條件句的前件蘊涵後件」，或者說「條件句的前件是後件的充分條件」，而雙條件句的意思，顧名思義，就是說「條件句的前件蘊涵後件，而且後件也蘊涵前件」，或者說「條件句的前件和後件是彼此的充分條件」。

充分條件與必要條件

前面提到條件句的意思就是在說條件句的前件會蘊涵條件句的後件，換句話說，條件句的前件會是條件句的後件之「充分條件」。而「充分條件」又和「必要條件」之間有著密切的關係，所以我們來花一些時間整理一下兩者之間的關係。

由於充分條件所表達的關係就是蘊涵關係，所以當我們假設 X 和 Y 分別是兩個任意語句時，說「X 是 Y 的充分條件」，意思就是在說「X 蘊涵 Y」，因此可知「只要 X 為真，則 Y 就一定為真。」我接下來會將此說法縮寫為「X 真 ⇒ Y 真」。

這一關係之所以會被稱為是「充分條件」的關係，是因為「充分」即是「足夠」的意思，因此當我們說「X 是 Y 的充分條件」時，意思就是 X 的成立就足夠讓 Y 也成立，而當我們說一個語句成立時，意思就是在說該語句為真。故「X 是 Y 的充分條件」的意思就是「X 為真能令 Y 也為真」。

那「必要條件」是什麼意思呢？讓我們一樣令 X 和 Y 是兩個任意的語句，由於「必要」是「必須」或「不可以沒有」的意思，因此當我們說「X 是 Y 的必要條件」時，意思就是 X 的成立對 Y 的成立而言是不可以沒有的。既然是不可以沒有的，意思就是要讓 Y 成立的話，不可以沒有 X 成立這個條件，更白話地說，就是 X 不先成立的話 Y 就不可能成立的意思。所以，當 X 是 Y 的必要條件時，意思就是「X 假 ⇒ Y 假」。

至於充分條件和必要條件之間的關係，就是當 X 是 Y 的充分條件時，Y 就會是 X 的必要條件；反過來說，當 X 是 Y 的必要條件時，Y 也會是 X 的充分條件。所以我們就稱「充分條件」和「必要條件」互為彼此的「逆關係」（converse relation）。同樣互為逆關係的關係還有「大於」和「小於」（「X 大於 Y」則「Y 小於 X」，「X 小於 Y」則「Y 大於 X」）、「父母」和「子女」（「X 是 Y 的父母」則「Y 是 X 的子女」，「X 是 Y 的子女」則「Y 是 X 的父母」）、買賣（「X 向 Y 買東西」則「Y 對 X 賣東西」，「X 對 Y 賣東西」則「Y 向 X 買東西」）等。

但為什麼「充分條件」和「必要條件」會是彼此的逆關係？我們可以試著用以下方式來證明：

讓我們先假設「X 是 Y 的充分條件」，若能利用此一假設來證明「Y 是 X 的必要條件」，那就證明了「只要 X 是 Y 的充分條件，則 Y 就會是 X 的必要條件」。那麼我們就先來證明此一方向。

假設「X 是 Y 的充分條件」（假設 1），根據假設 1 可知，「X 真 ⇒ Y 真」。由於我們想要證明的是「Y 是 X 的必要條件」，因此就是要證明「Y 假 ⇒ X 假」。而要邆明這件事，就等於要證明「假設 Y 假則 X 一定也為假」。所以我們就進一步假設「Y 為假」（假設 2）。因此接下來要考慮的問題就是，當假設 1、2 都成立時，X 是否一定為假？注意，由於 X 是一個語句，所以根據我們在前一章提過的二值性，X 只能是真的或假的，也就是說，X 若不為真就一定為假。所以要考慮 X 在假設 1 和 2 都成立的情況下是否一定為假，我們可以先考慮 X 在假設 1 和 2 皆成立的情況下是否有可能為真？若不可能為真，那就一定為假。

根據假設 1，我們知道「X 真 ⇒ Y 真」，那麼假如 X 為真的話，Y 就一定為真。但根據假設 2，Y 為假，所以在假設 2 成立的情況下 Y 不可以為真，那麼 X 便也不可以為真（否則根據假設 1，Y 就會為真）。因此在假設 1 和 2 皆成立的情況下，X 一定為假。也就是說，在假設 1 成立的情況下，只要 Y 為假則 X 也一定為假。因此我們就證明了「當 X 是 Y 的充分條件時，Y 就一定會是 X 的必要條件」。

至於令一個方向，也就是「只要 X 是 Y 的必要條件，則 Y 就會是 X 的充分條件」，也可以用一樣的策略來證明。

這次我們先假設「X 是 Y 的必要條件」（假設 3），而根據假設 3 可知，「X 假 ⇒ Y 假」。由於我們要證明的是「Y 是 X 的充分條件」，因此就是要證明「Y 真 ⇒ X 真」。而要邆明這件事，就等於要證明「假設 Y 真則 X 一定也為真」。所以我們就進一步假設「Y 為真」（假設 4）。因此接下來要考慮的問題就是，當假設 3、4 都成立時，X 是否一定為真？和剛才一樣，由於 X 是一個語句，所以根據我們在前一章提過的二值性，X 只能是真的或假的，也就是說，X 若不為假就一定為真。所以要考慮 X 在假設 3 和 4 都成立的情況下是否一定為真，我們可以先考慮 X 在假設 3 和 4 皆成立的情況下是否有可能為假？若不可能為假，那就一定為真。

根據假設 3，我們知道「X 假 ⇒ Y 假」，那麼假如 X 為假的話，Y 就一定為假。但根據假設 4，Y 為真，所以在假設 3 成立的情況下 Y 不可以為假，那麼 X 便也不可以為假（否則根據假設 3，Y 就會為假）。因此在假設 3 和 4 皆成立的情況下，X 一定為真。也就是說，在假設 1 成立的情況下，只要 Y 為真則 X 也一定為真。因此我們就證明了「當 X 是 Y 的必要條件時，Y 就一定會是 X 的充分條件」。

如此一來，我們就成功地證明了「充分條件」和「必要條件」確實互為彼此的逆關係。那麼，由於雙條件句所表達的意思是「條件句的前件與後件互為彼此的充分條件」，也就是說，雙條件句的前件是後件的充分條件，而後件也是前件的充分條件，故可知雙條件句的前件也會是後件的必要條件（因為後件是前件的充分條件），而後件也會是前件的必要條件（因為前件是後件的充分條件）。所以我們可以更完整地說，雙條件句要表達的意思是，條件句的前件和後件互為彼此的「充分且必要條件」。而這個「充分且必要條件」的關係會是雙向的關係（也被稱為對稱關係），有時候我們也會把它簡稱為「充要條件」。

當兩個語句互為彼此的充要條件時，這兩個語句就一定會同真同假（要嘛一起為真，要嘛一起為假，不可能一真一假）。這是因為當兩個語句互為充要條件時，由於它們一定互為充分條件，所以只要其中一個為真，另一個就一定也會真；同時它們也互為必要條件，因此只要其中一個為假，另一個也一定為假。所以當兩個語句互為充要條件時，它們就一定會同真同假。

而當兩個語句要嘛同樣為真、要嘛同樣為假時，通常都是因為這兩個語句所表達的是同一個意思，所以語句間互相蘊涵（或互為充要條件）的關係常被視為語意上相等的指標。但仍有些例外的情況必須注意，比如說當兩個語句都是恆真句或都是矛盾句時（恆真句和矛盾句的意思在之後的 3.2 中會說明），它們也會總是要嘛同樣為真要嘛同樣為假，但卻可能表達不同的意思。另外也有一些特別的情況是，兩個句子在談的雖然是同一件事，但語意上來說卻不相同，比如「杯子裡有水」和「杯子裡有H2O分子」就是在談同一件事，但「水」的意思和「H2O」在語意上卻沒有關係（否則十八世紀以前的人類都不知道「水」的意思）。不過由於上述這些案例都只算是特例，所以一般的情況下，我們還是可以認為兩個語句互相蘊涵（也就是互為充要條件）時，就意味著這兩個語句的意義是相等的。

以上就是關於充分條件與必要條件的說明，底下就可以回到語句符號和語句連接詞符號的討論。

在理解和使用這些語句連接詞時必須特別注意一點：由於語句邏輯的語言中能夠承載意義的最基本元素就是語句，所以這些語句連接詞所連接的當然都必須是語句，因此每一個「……」的位置都必須放上一個語句。說到這裡，有些眼尖的讀者可能已經發現了，在這五個連接詞當中，第一個只有連接一個語句，而剩下四個都各自連接了兩個語句。為了區別，我們就將一個連接詞依照它連接的語句數量，區分成一元或二元連接詞，所謂的「元」，指的就是一個連接詞能夠連接的東西的數量。

綜上所述，我們所使用的語句邏輯語言中主要有兩類元素，其中一類元素是「語句符號」，包括大寫的英文字母如$'A'$、$'B'$、$'C'$……等，當不夠用時，也可以使用加上了自然數下標的大寫英文字母如$'A_3'$、$'B_14'$、$'C_213'$……等，這些符號的功能就是用來代表語句（僅限定不可再被分割的簡單語句）。另外一類元素則是語句連接詞符號，一共有五個，其中有一個一元連接詞「並非……」，我們用$'¬'$這個符號來表示，還有四個二元連接詞：「……而且……」、「……或者……」、「如果……，則……」、「……當且僅當……」，我們分別以$'∧'$、$'∨'$、$'→'$、$'↔'$等四個符號表示之，而這些符號的功能就是用來將簡單的語句組成複雜的語句。然而除了這兩類元素之外，事實上還有第三類元素，我們稱之為「分組符號」，包括常見的左括號 '(' 和右括號 ')'，這些就是在整個語句邏輯語言中可以使用的所有符號元素。

用語句邏輯的語言來表達

語句連接詞的用法

要如何透過語句符號和語句連接詞符號來創造新的語句呢？我們先用中文裡的語句和語句連詞的關係來說明，例如：

  1. 海綿寶寶住在深海的大鳳梨裡。
這是一個有意義的中文語句，如果在它的前面加上一個「並非……」的語句連接詞，就會變成：
  1. 並非海綿寶寶住在深海的大鳳梨裡。
但這個寫法其實容易令人產生誤會，以為這句話是在說「住在深海的大鳳梨裡的不是海綿寶寶」，換句話說，這個寫法容易讓人誤以為「並非」在這個句子裡是用來否定「海綿寶寶」的。

可是要注意，當我在介紹「並非……」這個語句連接詞的時候，我有特別強調，這個連接詞（以及其餘的四個連接詞）所連接的都必須是「語句」，但「海綿寶寶」只是一個名詞而不是語句，因此不能是「並非……」這個連接詞所連接的東西。因此，在這個新語句中，「並非……」所否定的其實是「海綿寶寶住在深海的大鳳梨裡」這一整句話。因此，我們可以用以下的寫法來避免誤解：

  1. 並非「海綿寶寶住在深海的大鳳梨裡」。

當我們把 b 寫成 c 的樣子後，就能較清楚地看出來，這句話既不是在說「住在深海的大鳳梨裡的不是海綿寶寶」，也不是在說「海綿寶寶不住在深海的大鳳梨裡」，這句話的意思其實是在說「『海綿寶寶住在深海的大鳳梨裡』這句話是假的」。這三句話之間其實有著微妙的差別，我們首先來看頭兩句：

  1. 住在深海的大鳳梨裡的不是海綿寶寶。
  2. 海綿寶寶不住在深海的大鳳梨裡。

這兩句話乍看之下極其相似，但兩句話想要表達的內容其實有一點不同。以 d 來說，雖然以字面上來說，這句話看起來好像只是在表達海綿寶寶不住在深海的大鳳梨裡，也就是 e 的意思。但若仔細分析 d 的語句結構的話，不難發現 d 其實是一個標準的「主詞─述詞」結構的陳述句：「住在深海的大鳳梨裡的」是該語句的主詞，這個主詞指到了一個我們所未知的某個對象。而「不是海綿寶寶」這個述詞則是用來說明，無論「住在深海的大鳳梨裡的」指到的對象是誰，那個對象並不是海綿寶寶。

也就是說，d 雖然沒有明白地講出來，但其實預設了「有某個誰住在深海的大鳳梨裡」，而這是 e 沒有表達的內容。因為 e 的主詞直接就是「海綿寶寶」，而「不住在深海的大鳳梨裡」則是用來描述海綿寶寶的居住狀態的述詞，它沒有指到海綿寶寶以外的任何對象（無論已知或未知）。

儘管 d 和 e 在表達的內容上有上述的微妙差異，但這兩句話的任何一句如果為真的話，都可以令 a 為假。也就是說，這兩句話其實都可以用來否定 a，因此這兩句話都可以令 c 為真。但是要注意的是，儘管 d 和 e 都可以用來否定 a，但 d 和 e 兩句話並不等於 a 的否定。此間的差異可以考慮以下兩個句子：

  1. 派大星是粉紅色的。
  2. 派大星是天藍色的。

這兩句話其實可以相互否定，f 可以否定 g，g 也可以否定 f，但這兩句話卻不等於彼此的否定句。為什麼？要記得，語句連接詞所連接的必須是「語句」，也就是說，如果要取得一個語句的否定句，就一定要用否定連接詞（也就是「並非……」）來連接一個完整的語句。因此，f 的否定句就應該是：

  1. 並非「派大星是粉紅色的」。還記得為什麼要這樣寫嗎？就像之前要把 a 的否定句寫成 c 而不是 b 一樣，是為了避免誤解，因此特別用單引號將被否定的整個語句標示出來。

h 宣稱：f 是假的。而當 f 為假的時候，我們能不能知道派大星是什麼顏色的？不能。我們最多只能知道，當 f 為假時，派大星不是粉紅色的，至於派大星是什麼顏色，h 並沒有告訴我們。也就是說，如果只是給定 h（也就是 f 的否定句），我們不能確定派大星的顏色是粉紅色以外的什麼顏色，但 g 卻明白地宣稱派大星的顏色是天藍色的，因此 g 和 h 的意思顯然不一樣。

既然 g 和 h 的意思不一樣，儘管兩者都可以用來否定 f，但只有 h 才是 f 的否定句。同理可證：儘管 f 也可以用來否定 g，但 f 也不等於 g 的否定句。

在理解了「並非……」這個語句連接詞的正確用法之後，其它連接詞的用法只要比照相同的方式使用即可。不過，因為「並非……」是一個一元的語句連接詞，所以它只需要連接一個語句即可，但其餘的四個語句連接詞都是二元連接詞，所以其餘的四個連接詞都必須連接兩個語句。所以我們來增加一個新的語句：

  1. 派大星住在石頭屋裡。
因為 a 和 i 分別都是一個語句，所以我們可以用「……而且……」這個語句連接詞將兩個語句連接起來。並且，因為這個連接詞也是語句連接詞，所以為了避免誤解，我們一樣要將連接詞所連接的兩個語句用單引號包起來，以標示說它連接的分別是兩個完整的語句。這麼做之後，我們可以得到：

  1. 「海綿寶寶住在深海的大鳳梨裡」而且「派大星住在石頭屋裡」。

由於 f 和 g 也分別是兩個完整的語句，所以我們也可以用「……或者……」這個語句連接詞將兩者連接起來，得到：

  1. 「派大星是粉紅色的」或者「派大星是天藍色的」。
同理，因為 g 和 h 也分別是完整的語句，所以我們可以用「如果……，則……」這個語句連接詞將兩者連接起來，得到：
  1. 如果「派大星是天藍色的」，則「並非「派大星是粉紅色的」」。
最後，我們也可以用「……當且僅當……」連接 f 和 g 這兩個完整的語句來得到：
  1. 「派大星是粉紅色的」當且僅當「派大星是天藍色的」。
以上的 c、j、k、l、m 分別示範了我們要使用的語句邏輯系統中的五個連接詞該如何使用，在使用這些連接詞時，有兩個重點必須要注意：一、所有的連接詞都必須用來連接完整個語句；二、一元的連接詞就必須連接一個完整的語句，二元的連接詞則必須用來連接兩個完整的語句。只要把握住以上這兩點，就能正確地使用這些語句連接詞。

將中文語句翻譯至語句邏輯的語言

在確定了這五個語句連接詞的正確使用方法之後，我們接下來就要知道如何將這些語句用語句邏輯的符號來表示。在說明語句邏輯的語言中有哪些符號時，我說我們會使用大寫的英文字母（在不夠用時可以加上自然數下標）來代表個別的語句，因此，上面的 a-m 都可以分別用一個大寫英文字母來表示。

但是要注意的是，在 a-m 這十三個語句當中，有些語句其實是用語句連接詞將語句連接起來所形成的複雜語句，像這樣的語句如果也只是用一個大寫英文字母來表示的話，我們就無法辨識出這些語句的結構。然而語句的結構無論是在理解一個語句的意義時，還是在使用語句進行推論時，都扮演了極為關鍵的角色。所以為了能夠完全展示出每一個語句的結構，我們要規定：每個大寫英文字母都只能代表一個在結構上無法再被分割的原子語句。

什麼是原子語句呢？說一個原子語句在結構上不能再被分割又是什麼意思呢？以語句 l 為例，我們知道語句 l 是一個單條件句，它是用一個語句連接詞「如果……，則……」將兩個句子連接起來所形成的複雜語句，我們以大寫英文字母 $'G'$ 和 $'H'$ 來分別表示該語句的前件和後件，那就可以得到「如果$G$，則$H$」這樣的句子。

但我們也知道，$'H'$ 所代表的語句 h 其實也是由一個語句連接詞「並非……」連上一個語句 f 所形成的複雜語句，所以為了將 $'H'$ 的結構也表現出來，我們必須將它進一步改寫。讓我們用 $'F'$ 來表示語句 f，那 $'H'$ 就可以被改寫成「並非$F$」，而「如果$G$，則$H$」就可以被改寫成「如果$G$，則並非$F$」。最後，我們再將其中的語句連接詞都改成符號，就會得到：

  1. $G→¬F$

在 l' 中，無論是 $'G'$ 或 $'F'$ 之中都已經不再有任何語句連接詞的存在，因此不是是由某個語句連接詞連接其它語句所形成的複雜語句，這樣的語句就是在結構上無法再被分割的語句，因此我們稱之為原子語句。

由於語句邏輯中的每一個語句符號（也就是可以加上自然數下標的所有大寫英文字母）都只能用來表示一個原子語句，因此當我們要將某個自然語言中的語句（如中文語句）翻譯至語句邏輯的語言時，必須特別注意被翻譯的那個自然語言語句當中有沒有語句連接詞，如果有，就必須將其連接的語句挑出來，然後再看看那些語句當中還有沒有語句連接詞，直到被挑出來的語句中不再有任何連接詞後，才能用語句邏輯中的語句符號來代表那些語句。

另外要注意的一件事是，在建造一個語句邏輯中的複雜語句時，我們往往不會只使用到一個語句連接詞，例如 l' 就同時使用到了 $'→'$ 和 $'¬'$ 這兩個連接詞。雖然在某些情況裡，這些句子的寫法不至於產生誤解，比如在 l' 中，它要表達的意思就很清楚，$'¬'$ 連接的語句是 $'F'$，而 $'→'$ 連接的語句則是 $'G'$ 和 $'¬F'$。但在更多的時候，如果沒有清楚的標示，就會造成誤解。比如說，如果我們將 l 中的前後件對調會得到：

  1. 「並非「派大星是粉紅色的」」則「派大星是天藍色的」。
若我們再依照前面的方法將 n 翻譯至語句邏輯的語言中，會得到：
  1. $¬F→G$
但這個寫法就有可能產生誤解，因為我們將不能直接從 n' 的寫法看出來句首的 $'¬'$ 究竟是僅僅作用在 $'F'$ 這個原子語句上，還是作用在 $'F→G'$ 這個複雜語句上。為了避免這種誤解，我們就需要對五個語句連接詞的黏性有所規定。在語句邏輯中，我們對這五個語句連接詞的黏性大小規定如下：$'¬'$ > $'∧'$ = $'∨'$ > $'→'$ = $'↔'$。

規定了語句連接詞的黏性大小後，當一個語句中出現了多個語句連接詞的時，我們就可以根據連接詞的黏性大小來判斷誰可以先把相鄰的語句搶走。當一個語句連接詞的黏性愈大的時候，它就能愈優先搶走與它相鄰的語句中結構最簡單的語句，直到那個語句連接詞的每個位置都有語句相連之後，才會輪到黏性次強的連接詞去搶與它相連的語句。

以 n' 為例，這個句子裡出現了兩個語句連接詞，分別是 $'¬'$ 和 $'→'$，其中 $'¬'$ 的黏性大於 $'→'$，所以當我們要解讀 n' 時，$'¬'$ 會先把與它相鄰的語句中結構最簡單的語句搶走，並且，因為 $'¬'$ 是個一元連接詞，而且它所相連的語句是在它的右邊，所以它就只能搶走它右側與它相鄰的語句中結構最簡單的一個語句。

在 n' 中，位在 $'¬'$ 右側且與它相鄰的語句一共有兩個，分別是 $'F'$ 和 $'F→G'$，而在這兩個語句當中，$'F'$ 的結構比 $'F→G'$ 還要簡單，所以 $'F'$ 是在 '¬' 右側的語句中結構最簡單的，因此 $'¬'$ 就會把 $'F'$ 黏走。在 $'¬'$ 把 $'F'$ 黏走並形成 $'¬F'$ 後，才輪到 $'→'$ 來黏去其它的語句，而此時在它左、右兩側與它相鄰的語句分別都只剩下一個，左側的是 $'¬F'$，右側的是 $'G'$，所以 $'→'$ 就沒有什麼好選的了，只能分別黏住這兩個語句並形成 $'¬F→G'$。

然而，僅僅依賴黏性並不足以解決所有可能的誤解，比如說，考慮以下這個複雜的句子：

  1. 「派大星是紛紅色的」而且「派大星住在石頭屋裡」，或者「派大星是天藍色的」。
若要將 o 翻譯至語句邏輯的語言中，我們可以把它寫成（用 $'I'$ 來表示語句 i）：
  1. $F∧I∨G$
然而，這個句子裡出現在兩個語句連接詞 $'∧'$ 和 $'∨'$ 的黏性相等，所以無法判斷哪一個可以優先搶走與自己相鄰的語句，因此在這種情況下，我們就要使用左右括號 '('、')' 來手動指定語句的分配。以 o' 為例，因為它是對 o 的翻譯，而 o 是先用「……並且……」來連接 f 和 i，然後才用「……或者……」來連接「f 並且 i」和 g，所以在翻譯的時候，我們應該將 o' 寫成：
  1. $(F∧I)∨G$
透過括號的使用，我們在解讀 o' 的時候就知道要先將 $'F'$ 和 $'I'$ 分配給 $'∧'$，然後再將 $'(F∧I)'$ 和 $'G'$ 分配給 $'∨'$。

語句邏輯的完構式

後設語言和對象語言

在進入下一階段的討論之前，有一個重要的區分必須先向大家說明，就是後設語言 (meta-language) 和對象語言 (object language) 的區別。大家的母語應該都是中文，自小開始就浸淫在中文的語言環境之中，在耳濡目染之下自然而然地學會了如何使用中文。當大家開始要學習一個新的語言（比如說英文）時，往往也都會以中文作為基礎來學習。

比如說，第一次在學校上英文課時，老師會用中文向大家介紹英文的26個字母，用中文講解英文字母的寫法，也用中文說明英文字母的發聲發式。在教英文單字時，也會用中文來說明單字的意義；在教英文文法時，也是用中文來說明（例如）動詞變化的型態，或特定句型的使用方式。

在這種情況下，由於我們是用「中文」來談論「英文」，所以「英文」是我們談論的對象，而「中文」則是我們用來談論「英文」的語言。因此，我們將英文稱為對象語言，用來標示出被我們當成談論對象的語言是什麼；而中文則被稱作後設語言，用來表示我們在對象語言的後面，設定了可以用來討論它的語言。

因為我們可以區分對象語言和後設語言，所以我們可以用一種語言來討論另一種語言。我們可以用中文來討論英文，可以用英文來討論法文，這樣一來，即使在學習一個新的語言時，也不需要從頭學起，可以將新的語言用我們自己熟悉的語言來理解，幫助我們更順利地學習到新的語言。那麼，既然語句邏輯的語言也是一套語言，當我使用中文來和大家介紹語句邏輯的語言時，我同樣是把中文當成用來討論語句邏輯語言的後設語言，而語句邏輯的語言自然就是被討論的對象語言。

符號的「使用」與「提及」

不過後設語言和對象語言的區分不只可以讓我們用一個語言來談論另一個語言，這個區分甚至可以讓我們用同一個語言來談論它自己。比如說，在前一章裡我和大家介紹中文的一些特性，像是中文的組合性，我會把一些中文語句挑出來分析和討論，這種時候我就是在用中文談論中文本身。

但是，當我們在同一個語言討論它自己的時候，我們要怎麼區分哪些部分是後設語言，哪些部分又是對象語言呢？這時候我們就需要依賴一些符號工具來標示這兩者的差別。舉例來說，在上一章裡討論到中文裡能夠表達意義的最基本元素時，我寫了以下段落：

語言中的所有元素都是由符號組合而成的，以中文來說，每一個中文字都是一個符號。有些字本身就有自己的意義，比如說「天」、「地」、「雨」、「水」，但有一些字必須和其它的字組合在一起才具有意義，比如「葡」和「萄」就必須連在一起寫成「葡萄」才會有意義。
從這個段落應該不難看出來哪些地方的中文是被當成後設語言在使用，而哪些地方的中文是被當成對象語言來看待。比如說，當我談到中文裡有些字本身就有自己的意義時，我舉了「天」、「地」、「雨」、「水」等四個中文字作為例子，也就是說，「天」、「地」、「雨」、「水」等四個字就是被當成對象語言看待的中文字，而其它用來討論這四個字的中文則是被當成後設語言在使用。

為什麼能夠區分得出來兩者的差異呢？因為從前後文的脈絡可以明顯地看出來，當我寫到「天」、「地」、「雨」、「水」這四個中文字的時候，我並不是用使用這些中文字來表達意義，我只是要提及 (mention) 這些字的存在，並且描述這四個中文字有何重要的特性。在這個脈絡下，我僅僅是把它們看作四個符號，並且把大家的注意力導向這四個符號本身，我沒有用它們來來表達任何意義。

相對地，當我在段落中說明這四個中文字不需要和別的文字搭配，本身就可以表達意義時，我則是利用中文字來表達這些說明的內容。這時候我的重點就是這些中文字能夠表達的意義，而不只是把那些中文字看成一個一個符號而已，我需要大家把注意力放在這些文字表達的意義上。在這個時候，我就不僅僅只是在提及這些中文字符，而是直接在使用 (use) 這些中文字（來表達意義、傳遞內容）。

當我只是要提及一些符號（無論是出自哪個語言的符號）時，我通常會用中文裡的單引號 '「' 和 '」' 來標示出我想要提及的符號，這是因為引號在中文裡通常是被用來引述別人說過的話，而引述這種語言行為其實就是對符號的提及，而非直接使用那些符號。考慮以下這個例子：

科科對安安罵髒話說：幹你老師！
安安決定去跟老師告狀，科科跟了上去。
安安對老師說：老師，科科剛剛罵髒話，他說「幹你老師」。
老師決定處罰科科，科科這時卻抗議說：
安安剛才也說「幹你老師」，你也要處罰安安！
科科的抗議沒有道理，因為安安只是提及（或引述）科科剛才使用過的符號，他本身並沒有使用那些符號來表達意義，也就是說，安安並沒有說髒話。（同樣的道理，當科科在抗議時提到安安剛才也說了「幹你老師」時，他也只是提及那些符號，而沒有用那些符號來表達意義，所以科科在那個當下也沒有說髒話。）

語句邏輯中的完構式

在區分過對語言的提及和使用的差異，並以此區分來分辨對象語言和後設語言的差別後，我們再回過頭看看上一節中我向大家介紹語句邏輯的語言時，是如何使用語言的。在上一節中，我是用中文向大家介紹語句邏輯有哪些符號可以使用，不同符號的功能是什麼，以及這些符號要如何構造出語句邏輯中的各種表達式。

當我要用中文來談到語句邏輯中的符號或表達時，我都會用英文的單引號（''）來將語句邏輯的符號標示出來，這表示我只是在提及那些符號，而非直接使用那些符號來表達意義。雖然那原本是英文裡的標點符號，但當我把這些符號和中文一起用來討論語句邏輯時，那這些英文標點符號就同時也是我用來討論語句邏輯的後設語言。也就是說，後設語言可以不只是一個單一的語言，當有需要的時候，我們就可以用多於一種的語言來討論其它語言。

現在，為了要說明以下的事情，我需要在我們用來討論語句邏輯的後設語言裡新增更多符號。在介紹語句邏輯語言中的符號時，我說我們會用大寫的英文字母（加或不加數字下標）來代表一個原子語句（就是當中不包含任何語句連接詞的語句）。但有時候我們需要討論到語句邏輯中的複雜語句，此時若沒有專用的符號來代表它們，討論就會變得很不方便，這時候就可以在後設語言裡加入某些符號專門用來代表語句邏輯中的複雜語句。

以下，我將用小寫的希臘字母如 $'α'$、$'β'$、$γ'$ 等來代表任何語句邏輯的語言可以表達的語句，這些符號不僅可以用來代表原子語句，也可以用來代表語句邏輯中的複雜語句。比如說，我們可以用 $'α'$ 來代表 $'A'$，用 $'β'$ 來代表 $'¬A'$，用 $'γ'$ 來代表 $'(F∧I)∨G'$。只是在這樣使用的時候，需要特別注意的是，這些符號都只是用來討論語句邏輯時的後設語言裡的符號，它們不屬於語句邏輯的語言，不能用來建構語句邏輯的表達式。它們只能用來談論到語句邏輯中的表達式。

在準備好這些可以用來代表語句邏輯中任何表達式的符號之後，我要介紹的是在語句邏輯中建構出任何合法語句時必須遵守的規則，而我們則稱所有語句邏輯中的合法語句為完構式 (well-formed formula)，意思是，這些表達式都是結構上完整的表達式。以下就是建構出語句邏輯中的完構式的所有規則：

  1. 所有語句符號（用來代表原子語句的符號）都是完構式。
  2. 如果 $α$ 和 $β$ 都是完構式，
    則 $'¬α'$，$'(α∧β)'$，$'(α∨β)'$，$'(α→β)'$，$'(α↔β)'$ 也都會是完構式。
  3. 只有用前二條規則逐步建構出來的表達式是完構式。
這三條規則除了規定我們該如何建構出語句邏輯的完構式之外，其實同時也定義了所有語句邏輯中的完構式會長什麼樣子。規則 1 規定了語句邏輯中最基本的完構式都是由單獨的一個語句符號所構成的，接著規則 2 則規定了如何利用語句連接詞將較簡單的完構式組合起來建構出較複雜的完構式，最後，規則 3 則宣告了只有透過規則 1 和 2 逐步建構出來的表達式才是完構式，因此宣告了其它方法都不可能建構出完構式。

實際應用這三條規則來試著建構出各種完構式，是熟悉這些規則最快的方法。比如說，$'A'$ 根據規則 1 是一個完構式，所以 $'¬A'$ 根據規則 2 也是一個完構式。而 $'B'$ 和 $'C'$ 根據規則 1 都是完構式，因此 $'(B→C)'$ 根據規則 2 也是一個完構式。那麼，由於 $'¬A'$ 和 $'(B→C)'$ 都是完構式，故再根據規則 2，$'(¬A∧(B→C))'$ 也會是一個完構式。而 $'D'$ 和 $'E'$ 也都是完構式（根據規則 1），所以 $'(D↔E)'$ 根據規則 2 是完構式，因此 $'((¬A∧(B→C))∨(D↔E))'$ 根據規則 2 也會是一個完構式。

在上述的應用過程中有兩個必須注意的重點。首先要注意到的是，規則 1 和 2 都是關於如何建構完構式的規則，但規則 1 是直接指出什麼東西是完構式，而規則 2 則是說明如何使用語句連接詞將較簡單的完構式組合成較複雜的完構式。而規則 3 則是要求所有完構式都必須是逐步利用規則 1 和 2 建構出來的，因此每個複雜的完構式的最後一個建構步驟，一定都是應用了規則 2 之中的某個句型來完成的。知道為什麼不會是應用了規則 1 建構出來的嗎？因為這邊談到的是「複雜的完構式」，如果一個完構式是根據規則 1 建構出來的，那它必定是一個原子語句，因此是結構最簡單的完構式，故不可能是一個複雜的完構式。因此，任何一個複雜的完構式的最後一個建構步驟一定都是用一個語句連接詞連接一個（根據規則 2 中的否定句型）或兩個（根據規則 2 中的其餘四個句型）建構出來的。

比如說，$'¬A'$ 就是根據規則 2 用 $'¬'$ 連接 $'A'$ 這一完構式所建構出來的完構式，而 $'(B→C)'$ 則是根據規則 2 用 $'→'$ 連接 $'B'$ 和 $'C'$ 兩個完構式所建構出來的完構式，而 $'(¬A∧(B→C))'$ 則是根據規則 2 用 $'∧'$ 連接 $'¬A'$ 和 $'(B→C)'$ 這兩個完構式所建構出來的。至於 $'((¬A∧(B→C))∨(D↔E))'$ 則是根據規則 2 用 $'∨'$ 來連接 $'(¬A∧(B→C))'$ 和 $'(D↔E)'$ 這兩個完構式建構出來的。

由於每一個複雜的完構式都有這樣的特性，因此我們一定可以在每一個複雜的完構式中，挑出建構它的最後一個建構步驟使用的語句連接詞。比如說，$'¬A'$ 的最後一個建構步驟使用到的語句連接詞是 $'¬'$；$'(¬A∧(B→C))'$ 的最後一個建構步驟使用到的連接詞是 $'∧'$；而 $'((¬A∧(B→C))∨(D↔E))'$ 的最後一個建構步驟使用到的連接詞則是 $'∨'$。我們統一將所有完構式的最後一個建構步驟使用到的連接詞稱為該完構式的主要連接詞。辨識出一個完構式（往後我將交替使用「完構式」和「語句」這兩個名詞）的主要連接詞對於判斷它的語法結構和語意都是必要的，這些在下一小節和下一章中會做更詳細的說明。

第二個重點則是括號的使用時機。請大家注意到規則 2 中對五種複雜語句句型的寫法，不難發現其中只有否定句 $'¬α'$ 並未在外側加上括號，其餘的四種句型則都有在外側加上括號。這條規則的寫法就清楚地規定了：只有當我們要用到四個二元連接詞來建構複雜語句時必須在建構好的語句外側加上括號，而在使用一元連接詞來建構複雜語句時則不用。比如說，在利用規則 2 建構出 $'(¬A∧(B→C))'$ 時，我們是先用規則 2 分別建構出 $'¬A'$ 和 $'(B→C)'$ 後，才將這兩個語句組成 $'(¬A∧(B→C))'$ 的。從上述的過程可以發現，由於 $'¬A'$ 是用一元連接詞加上一個原子語句 $'A'$ 所組成的，所以其外側並未加上括號，而 $'(B→C)'$ 則是用一個二元連接詞連接兩個原子語句 $'B'$ 和 $'C'$ 所組成的，因此外側就要加上括號。最後，當我們用 '∧' 將 $'¬A'$ 和 $'(B→C)'$ 組合起來時，由於 $'∧'$ 也是一個二元連接詞，因此最後建構出來的 $'(¬A∧(B→C))'$，其外側也加上了括號。

之所以會在完構式的建構規則中設計這些括號的使用方式一共有兩個目的，其一是為了確保語句邏輯中的完構式只能有唯一的語意，而當一個表達式中存在兩個以上的語句連接詞時，就有可能出現某個語句有多種解讀可能的情況。比如說，令 $'A'$ 表示「小明會說中文」、$'B'$ 表示「小明會說英文」、$'C'$ 表示「小明會說日文」。當我們將這三個原子語句組成 $'A∧B∨C'$ 時，由於沒有括號，所以無法確定這個句子究竟該讀成 $'A∧(B∨C)'$ 還是 $'(A∧B)∨C'$ 兩種讀法，而且這兩種讀法下的語句意思會完全不同。若採取前一種讀法，則該語句表達的意思是，小明至少會說兩種語言，其中一種是中文，而另一種則可能是英文或日文；但若採取後一種讀法，則該語句的意思是，小明有可能既會說中文也會說英文，但也有可能小明只會說日文。所以，為了讓表達式只有一種正確的讀法，我們就必須利用括號將語句用恰當的方式組合起來，以確保不會產生誤會。

然而確保語句有唯一的解讀方式並不是括號唯一的目的，使用括號的另一個目的是為了確保每一個語句在語法上的正確性。比如 $'A∧B∧C'$，它固然也可以用兩種不同的方式來置放括號：$'A∧(B∧C)'$ 或 $'(A∧B)∧C'$，但無論將括號置於 $'A∧B'$ 的外側還是置於 $'B∧C'$ 的外側，讀起來的意思都一樣，都是表達了小明三種語言都會說的意思。因此是否我們就不需要為 $'A∧B∧C'$ 上括號？答案是否定的，因為若不加上括號，這個語句在語法上就會有問題。

什麼是語法？

什麼是語法問題？所有的語言其實都可以區分出語法（syntax）和語意（semantics）兩個面向。語言說穿了就是一套符號系統，當我們操作符號時，我們總是會有兩種不同面向的關懷：首先，我們會在意每一個符號代表的意思是什麼？也會在意不同的符號組合在一起後是否會表達出新的意思？這種關懷就是對語意面向的關懷。但除此以外，我們也會在意每一個符號的寫法和長相是什麼？也會在意自己對符號的操作與排列組合是否正確或恰當？而當我們單純在關注語法問題時，是可以完全不管語意面向的。

這麼說可能會讓多數人覺得有點奇怪，首先，我們在使用語言的時候確實既會在意符號的意思，也確實會在意符號的寫法與排列組合和操作方式。但即使拿最簡單的一條文法「主詞+動詞+受詞」來看的話也會發現，就算是對詞性的區分（誰是主詞？誰是動詞？誰是受詞？）也需要依賴我們對語詞意義的理解。這樣看來似乎無法單純地根據符號的長相來決定正確的排列方式。這觀察確實是正確的，在我們的日常語言裡確實很難將語法和語意的面向清楚地區分開來，但仍然會有一些例子可以用來說明語法和語意的差異。

一個重要的語言學家和哲學家喬姆斯基（Noam Chomsky）曾經寫下過這樣一個句子：「無色的綠色觀念狂亂地睡著」。這句話在文法（特別是不同詞性之語詞的組合方式）上是正確的，先用形容詞「無色的」和「綠色」來描述一個主詞「觀念」，再以一個副詞「狂亂地」來修飾動詞「睡著」，但我們仍會認為要理解這個句子會有困難。首先是用來描述主詞的兩個形容詞在語意上是互相矛盾的（無色的就不可能是綠色，而綠色的就不可能無色）；再者，主詞是一個抽象名詞，因此也不可能有顏色，因為只有具體的東西才可能有顏色；接著，睡著通常是生物才會擁有的行為，而「觀念」顯然不是生物，也就不可能睡著；最後，睡著至少在外觀上通常是平靜的，而不會用「狂亂」來去修飾。因此基於這些語意上的衝突和詭異，我們才會認為這個合乎文法的句子卻是難以理解的。

因此即使語意和語法的面向在日常語言中難以被清楚區別開來，但透過這個例子我們還是能夠發現，即使在語法（或文法）上正確的語句，也可能是無法理解（沒有語意）的。至於在邏輯語言這種人工語言中，由於所有符號的功能都是被明確區分開來的，所以語意和語法的區隔就更明顯。我們完全可以不去問符號代表的意思，也能夠單純根據符號的長相來決定怎樣的語句是格式正確的。我們在上面介紹的完構式規則就是一種純粹的語法規則。

解釋完「語法」這一概念，讓我們回到括號能夠確保語句在語法上的正確性這件事。為什麼缺少括號就無法確保一個語句在語法上的正確性？我們繼續用 $'A∧B∧C'$ 這個例子來說明。根據前面的討論，我們發現這個表達式若要加上括號的話可以加在前面的 $'A∧B'$ 外側，也可以加在後面的 $'B∧C'$ 外側，但無論將括號加在哪裡，意思都是一樣的。但如果因此就不加括號的話，語法上會産生的問題是：這個表達式中用到了兩次連接詞 '∧'，而連接詞 '∧' 是一個二元連接詞，因此使用兩次的話就需要連接一共四個語句。但在 $'A∧B∧C'$ 中能夠表示語句的符號只有三個（$'A'$、$'B'$、$'C'$），因此不夠兩個二元連接詞分配。但只要加上括號，無論是寫成 $'A∧(B∧C)'$ 還是 $'(A∧B)∧C'$，都會因為括號將表達式中的一部分組合起來而形成一個新的語句，使得兩個二元連接詞都有剛好兩個語句可以連接，因此才成為語法上正確的語句。

我們就用 $'A∧(B∧C)'$ 這個寫法來做更詳細的說明。在此寫法中，右側的 '∧' 先是連接了 $'B'$ 和 $'C'$ 這兩個語句，然後再透過一組括號將 $'(B∧C)'$ 變成了一個新的語句，然後左側的 '∧' 才又將 $'A'$ 和 $'(B∧C)'$ 連接在一起。這樣一來才使得兩個二元連接詞各自都有兩個語句可以連接。

只不過，當一個語句愈寫愈長時，需要用到的括號就會愈來愈多，當括號多到一個程度時，有時候反而會令人看了眼花繚亂、妨礙閱讀，所以我們也會允許在一些不影響語意的情況下適當地省略括號。首先，我們會發現只要在建構語句時最後一步用到的是二元連接詞的句型（包括連言句、選言句、條件句、雙條件句）時，整個語句的最外層一定會有一組括號，但這組括號無論在語意上和語法上其實都沒有功能，所以只要在寫好一個完整語句後發現整個語句的最外層有一組括號的話，那麼最外層的這組括號是可以省略的。例如我們在前一段討論的 $'A∧(B∧C)'$，如果要用完全符合規則的方式來寫的話其實應該寫成 $'(A∧(B∧C))'$，但是因為最外層的那組括號在語法和語意上都沒有功能，所以就被省略了。

但在省略語句最外層的括號時需要注意的是，只有當一組括號的外側完全沒有任何符號時才算是真正最外層的括號，也只有真正最外層的括號才能省略。舉例來說，當我們要表達 $'(A∧(B∧C))'$ 的否定句時，就要在這個語句的最左側加上一個否定號，寫成：$¬(A∧(B∧C))$。在這個語句中我們可以看到兩組括號，一組是 $'B∧C'$ 外側的括號，另一組則是 $'A∧(B∧C)'$ 外側的括號。由於整個語句一共只有這兩組括號，而前一組又被包含在後一組括號中，所以單就這兩組括號的內外層來比較的話，我們可以說後一組括號是「（語句中的所有括號中）最外層」的括號，但即使後一組括號確實是（語句中的所有括號中）最外層的括號，但因為該組括號的外側仍有其它符號（即否定號 '¬'），所以它仍不算是「真正最外層」的括號，故不能省略。

我們也可以嘗試省略該組括號來看看會産生的影響。當我們將 $'¬(A∧(B∧C))'$ 中的所有括號中最外層的括號省略後，會變成 $'¬A∧(B∧C)'$。然而根據我們先前介紹過的五個連接詞的黏性差異，'¬' 是所有連接詞中黏性最大的連接詞，因此它會最優先將其右側結構最簡單的語句連接走，所以 $'¬A∧(B∧C)'$ 就應被解讀成用 '∧' 來連接 $'¬A'$ 和 $'(B∧C)'$ 這兩個語句而形成的連言句，而不再是用 '¬' 來連接 $'(A∧(B∧C))'$ 所組成的否定句，因而變成意思完全不一樣的語句。這就是為什麼即使它是語句中的所有括號中最外層的括號，但由於該組括號的外側仍有其它符號，所以仍然不能被省略的理由。

除了整個語句中真正最外層（也就是其外側不再有其它符號）的括號可以省略以外，我們也可以利用語句連接詞的黏性差異來省略括號。以 $'(¬A∧(B→C))'$ 這個語句來說，它是以 $'∧'$ 這個語句連接詞連接 $'¬A'$ 和 $'(B→C)'$ 所組成的，由於 $'¬A'$ 中的連接詞 $'¬'$ 的黏性大於 $'∧'$，所以即使不在 $'¬A'$ 的外面加上括號，$'A'$ 也不會被 $'∧'$ 黏走，因此在規則的設計上就直接讓否定句 $'¬α'$ 無須加上括號。但因為 $'→'$ 的黏性比 $'∧'$ 小，所以 $'(B→C)'$ 外側的括號就不能省略，否則 $'B'$ 就會被 $'∧'$ 黏走，而讓建構出來的語句變成不同的語句。如果不在 $'B→C'$ 的外面加上括號就寫進語句裡的話，該語句會變成 $'¬A∧B→C'$，由於這個語句中的三個語句連接詞的黏性都不一樣，所以這個語句即使不加上任何括號也只有一種讀法，只是不加括號的讀法會和前面的寫法不同。請你試試看用括號來標示出 $'¬A∧B→C'$ 的唯一讀法會是什麼？

相對地，當我們將例子換成 $'(¬A→(B∧C))'$ 時，這就變成是用連接詞 '→' 來連接 $'¬A'$ 和 $'(B∧C)'$。其中的 $'¬A'$ 由於是否定句，所以依然不用加括號，而右側的 $'(B∧C)'$ 則因為是連言句，所以必須加括號。但實際上 '∧' 的黏性是比 '→' 更大的連接詞，故若省略 $'(B→C)'$ 外側的括號後，寫成的 $'¬A→B∧C'$，仍會先由 '∧' 連接了 $'B'$ 跟 $'C'$（以及由 '¬' 連接了 $'A'$），再由 '→' 連接 $'¬A'$ 和 $'B∧C'$。既然意義仍然相同，且語法上也能分辨得出哪個部分先被組成了一組語句，因此這樣的括號也被允許省略。

只要讀者們能適當地搭配使用上述提到的兩條不成文的括號省略原則（真正最外層的括號可省略；可依據連接詞的黏性大小省略括號），就能在同時確保了語句的意義唯一性和語法正確性的條件下，寫出簡潔的完構式。但對初學者來說，我會建議在你真正熟悉了語句邏輯的語法規則以前，盡量仍使用完整的括號，最多只省略真正最外層的括號就好，否則將容易在建構語句的時候出錯。

判斷一個表達式是否為完構式

在上一小節裡，我們介紹了語句邏輯中的完構式（也就是語句）的建構方式。在這一小節中，我將說明如何判斷一個已經寫好的語句邏輯表達式究竟是不是一個完構式。

要判斷一個語句邏輯的表達式是不是一個完構式，首先要知道如何判斷一個表達式的主要連接詞。只要一個表達式是完構式，它就一定會有一個主要連接詞，而且這個主要連接詞必定是唯一的。我在上一小節中有提到，任何複雜語句的建構方式都是逐步利用某個語句連接詞來連接一個或兩個語句（視使用的連接詞是一元或二元連接詞而定），因此每一個複雜語句一定都會有一個連接詞是在建構該語句的最後一個步驟時用到的連接詞，那個連接詞就會是這個複雜語句中的主要連接詞。

假如這個語句本就是你自己建構出來的，你自然知道自己最後一個步驟用到的連接詞是哪一個，但假如你今天面對的是一個別人寫好的表達式，那麼要如何判斷出最後一步用到的連接詞就會需要一些技巧。不過為了要介紹判斷的技巧，我們需要先確定如何為表達式中的括號進行分層：

首先，在任何完構式中，每組括號一定都會剛好包含一個左括號和一個右括號，括號的使用一定都是成對使用的，不可能出現只用一個左括號或只用一個右括號的情況，因此任何一個完構式的左、右括號數量一定相同。

接著，在一個完構式中，任兩組括號（所謂一組括號就是一對左、右括號的組合）之間只會有兩種關係，一是一組括號包含另一組括號於其中，而被包含的那組括號的左、右括號一定都會被夾在另一組的左、右括號之間（所以順序上會呈現為左、左、右、右）；二則是兩組括號互不包含，因此也就不會有某一組的左、右括號都被夾在另一組的左、右括號之間（所以順序上會呈現為左、右、左、右）。

最後，當我們在為一個完構式的括號進行分組時，一律都是從最內層的括號開始分組，然後再一層一層地往外推。所以首先一定是把所有緊鄰的左、右括號都配對後，忽略這些剛配對好的括號，繼續去找下一批左、右括號彼此緊鄰的組合，如此直到所有括號都被配對完成為止。

以 $'(¬(A→B)∧((B↔C)∨¬C))'$ 為例，在分組這一表達式的括號時，我們需要先把其中彼此緊鄰的左、右括號組合都找出來。因此會先找到的是在 '∧' 左側之 $'¬(A→B)'$ 中的 $'(A→B)'$ 的這組括號，以及在 '∧' 右側之 $'((B↔C)∨¬C)'$ 中的 $'(B↔C)'$ 的這組括號，那麼這兩組括號便是這個語句中最內層的括號，我們將之稱為第 1 層括號。

接著要繼續找到下一層的括號，我們就必須暫時忽略剛才已經配對好的那兩組括號（當作他們不存在），再繼續找到下一批彼此緊鄰的左、右括號組合，也就是要當作我們在找 $'(¬\cl "grey"(A→B\cl "grey")∧(\cl "grey"(B↔C\cl "grey")∨¬C))'$ 中最內層的括號（請假裝顏色較淡的括號不存在）。此時我們會找到的就是在 '∧' 右側的 $'(\cl "grey"(B↔C\cl "grey")∨¬C)'$ 的這組括號（若將顏色較淡的括號視為不存在的話，則顏色正常的那對左、右括號就會是彼此緊鄰的左、右括號）。而這就是例句的第 2 層括號。

最後，我們再將剛才找到的括號也當成不存在的話，那下一批可以找到的彼此緊鄰的左、右括號就會是 $'(¬\cl "grey"(A→B\cl "grey")∧\cl "grey"(\cl "grey"(B↔C\cl "grey")∨¬C\cl "grey"))'$ 中剩下的唯一一組（顏色正常的）左、右括號，這組括號因此就會是例句的第 3 層括號。由於這組括號已經沒有被任何括號包含於其中了，所以我們也可稱之為例如的最外層括號。不過要注意的是，此時所謂的「最外層」僅只是指「語句中的所有括號中最外層的」括號而已，並不一定是語句中「真正最外層」的括號，因此是否能夠省略尚需進一步的判斷。對這個例句來說，它的最外層括號外側已經沒有任何其它符號了，因此確實也符合「真正最外層」括號的條件，所以是可以省略的括號。

知道如何為表達式中的括號分組後，要判斷出表達式的主要連接詞就變得很容易。要找出一個表達式的主要連接詞，我們首先要做的就是找出這個表達式的最外層括號。找到表達式的最外層括號後，此時會分成三種情況：一、這組最外層括號同時也是真正的最外層括號，也就是它的外側完全沒有其它符號；二、這組最外層括號的外側雖然有其它符號，但只有左側有，右側沒有，且左側只有出現否定連接詞 '¬'；三、這組最外層的括號外側不但有其它符號，而且並非只有左側有符號，或者雖然只有左側有符號，但左側不只有出現否定連接詞 '¬'，又或是整個表達式根本就沒有任何括號的情況。

在第一種情況下，也就是表達式的最外層括號就是真正的最外層括號時，我們就要找出在這組括號中且只在這組括號中唯一黏性最小的二元連接詞，這個連接詞就會是表達式的主要連接詞。首先，所謂「在這組括號中且只在這組括號中」的意思是，你要找的就是這組括號中的連接詞，假如這組括號內還包含了其它括號在其中，那些括號中出現的連接詞都不是我們要找的對象。

就以 $'(¬(A→B)∧((B↔C)∨¬C))'$ 這個例句來說，該表達式中最外層括號內還有另外三組括號（且其中一組將另一組包含在其中），因此出現在那三組括號中的連接詞都不用考慮，所以我們只需要考慮直接出現在最外層括號中的 '¬' 和 '∧' 就可以了。而在前述這兩個連接詞中，只有 '∧' 是一個二元連接詞，所以它自然就會是唯一黏性最小的二元連接詞，因此它就會是這個表達式的主要連接詞。

接著讓我們考慮另一個例子，比如： $'(A→B∧¬C)'$。由於此表達式中只有出現一組括號，而這組括號也確實是表達式中真正最外層的括號，所以我們要考慮的就是這組括號中的所有連接詞（因為這組括號中並未包含其它組括號），也就是（依左至右的順序）'→'、'∧'、和 '¬' 這三個連接詞。而在這三個連接詞中只有兩個是二元連接詞，也就是左邊的 '→' 和中間的 '∧'。而且在這兩個連接詞中，'→' 的黏性又比 '∧' 更小，所以 '→' 就是這個表達式的最外層括號中唯一黏性最小的二元連接詞，因此它就是這個表達式中的主要連接詞。

接著再看一個我們在前一小節中使用過的例子：$'(A∧B∨C)'$，這個表達式的最外層括號也是真正最外層的括號，且其內部也未包含其它括號，所以這組括號內的所有連接詞都是我們要考慮的連接詞，也就是左邊的 '∧' 和右邊的 '∨'。然而這兩個連接詞不但都是二元連接詞，而且兩個連接詞的黏性是相同的，所以在這個表達式的最外側括號中就不存在「唯一黏性最小的二元連接詞」，因此這個表達式就 沒有主要連接詞。注意，由於複雜完構式一定會有主要連接詞，所以既然這個複雜的表達式沒有主要連接詞，這個複雜的表達式就不是一個完構式。

在第二種情況下，也就是表達式的最外層括號的外側還有其它符號，但只有左側有，而且只有否定連接詞符號。例如，在 $'¬(A→B)'$ 這個表達式中的最外層括號外側尚有其它符號的存在，但只有左側有、右側沒有，且左側只有一個否定連接詞 '¬'。此時這個否定連接詞就會是這個表達式的主要連接詞。

不過第二種情況的條件所允許的還有另外一類表達式，像是 $'¬¬¬(A→B)'$，在此表達式中的最外層括號的外側仍有其它符號，且只有左側有、右側沒有，而且左側只有出現否定連接詞，只是此時不只有一個否定連接詞符號，而是三個！但無所謂，只要表達式的最外層括號外只有左側有別的符號，而且只有否定連接詞，那麼無論出現多少個，一律都是由最左邊的否定連接詞擔任表達式的主要連接詞。

至於在第三種情況下，也就是表達式的最外層括號外並非只有左側有其它符號，或者雖然只有左側有其它符號，卻不是只有否定連接詞，又或者是整個表達式根本就沒有任何括號的情況。比如像 $'(A→B)∨C'$ 這種只有最外層括號的右側有符號的、或是像 $'A∧B↔(A∨B)∧¬C'$ 這種最外層括號的左右兩側都有符號的、又或是像 $'A∧B→(C↔D)'$ 這種雖然只有在最外層括號的左側有符號，但卻不是只有否定連接詞的、又或是像 $'¬A∨¬B'$ 這種完全沒有任何括號的表達式，全都算在第三種情況下。而這種情況下的表達式主要連接詞的判斷方式也很簡單，只要為整個表達式加上一組真正最外層的括號，然後比照第一種情況來處理即可。

因此像上一段中的第一個例子 $'(A→B)∨C'$，為它加上一組真正最外層的括號後會得到 $'((A→B)∨C)'$，而在後面這個表達式中，直接在真正最外層的括號中出現的連接詞只有一個 '∨'，它也確實是一個二元連接詞，因此它就是此一表達式的主要連接詞。再以 $'A∧B↔(A∨B)∧¬C'$ 為例，為它加上一組真正最外層的括號後會得到 $'(A∧B↔(A∨B)∧¬C)'$，而在此表達式的真正最外層括號中直接出現的連接詞共有（左至右）'∧'、'↔'、'∧'、'¬' 這四個，而在這四個連接詞中只有前三個是二元連接詞，且其中又以 '↔' 為唯一黏性最低的連接詞，因此這就是該表達式的主要連接詞。以此類推， $'A∧B→(C↔D)'$ 的主要連接詞就會是左數第二個連接詞的 '→'；而 $'¬A∨¬B'$ 的主要連接詞就會是唯一的 '∨'。

介紹完任意表達式的主要連接詞之判斷方式後，接著就來說明如何判斷一個表達式是不是完構式。判斷的方式其實相當簡單，只要根據以下兩個步驟不斷地操作就能得到答案：

  1. 判斷表達式是否為原子語句（只有一個語句符號且沒有任何連接詞）？
    是 ⇒ 是完構式；
    不是 ⇒ 前進至步驟 2
  2. 判斷表達式的主要連接詞。
    無 ⇒ 不是完構式；
    有 ⇒ 是一元連接詞，則將其右側的整個表達式截取出來；
    有 ⇒ 是二元連接詞，則將其左右的表達式都分別截取出來。
    有 ⇒ 再針對截取出來的表達式從步驟 1 開始做。
在輪流執行上述兩個步驟的過程中，只要其中有任何一個環節發生找不到主要連接詞的情況，都可能直接斷定表達式不是完構式。相對地，假如能對一個表達式反複執行上述兩個步驟直到最後都只剩下原子語句時，就可以斷定表達式是一個完構式。

我們就先以 $'A∧B↔(A∨B)∧¬C'$ 這個表達式為例，來示範如何判斷它是不是一個完構式：

  • 首先可判斷出這不是一個原子語句，故前進至步驟 2。
  • 判斷其主要連接詞為 '↔'，是二元連接詞，故將其左右的表達式都截取出來。分別是 $'A∧B'$ 和 $'(A∨B)∧¬C'$。
    再分別對以上兩個表達式從步驟 1 開始做。
    • 先確認 $'A∧B'$ 不是原子語句，故前進至步驟 2。
    • 判斷其主要連接詞為 '∧'，是二元連接詞，故將其左右的表達式都截取出來。分別是 $'A'$ 和 $'B'$。
      再分別對以上兩個表達式從步驟 1 開始做。
      • 先確認 $'A'$ 為原子語句，故 $'A'$ 為完構式。
      • 再確認 $'B'$ 為原子語句，故 $'B'$ 為完構式。
    • 再確認 $'(A∨B)∧¬C'$ 不是原子語句，故前進至步驟 2。
    • 判斷其主要連接詞為 '∧'，是二元連接詞，故將其左右的表達式都截取出來。分別是 $'(A∨B)'$ 和 $'¬C'$。
      再分別對以上兩個表達式從步驟 1 開始做。
      • 先確認 $'(A∨B)'$ 不是原子語句，故前進至步驟 2。
      • 判斷其主要連接詞為 '∨'，是二元連接詞，故將其左右的表達式都截取出來。分別是 $'A'$ 和 $'B'$。
        再分別對以上兩個表達式從步驟 1 開始做。
        • 先確認 $'A'$ 為原子語句，故 $'A'$ 為完構式。
        • 再確認 $'B'$ 為原子語句，故 $'B'$ 為完構式。
      • 先確認 $'¬C'$ 不是原子語句，故前進至步驟 2。
      • 判斷其主要連接詞為 '¬'，是一元連接詞，故將其右側的表達式截取出來。是 $'C'$。
        再對以上表達式從步驟 1 開始做。
        • 先確認 $'C'$ 為原子語句，故 $'C'$ 為完構式。
到此為止，整個表達式就再也沒有任何東西可以繼續分析了，而且在判斷的過程中也沒有在任何步驟發現表達式的任何部分沒有主要連接詞，如此就能確定整個表達式確實是一個完構式。

相反地，假如在用上述方法判斷一個表達式是否為完構式時，有在任何一個步驟發現從表達式中截取出來的任何部分是沒有主要連接詞的，則該表達式就整個都不是完構式。讓我們以 $'(A∧(B↔C→D))'$ 為例來示簵：

  • 首先判斷出這不是一個原子語句，故前進至步驟 2。
  • 判斷其主要連接詞為 '∧'，是二元連接詞，故將其左右的表達式都截取出來。分別是 $'A'$ 和 $'(B↔C→D)。
    再分別對以上兩個表達式從步驟 1 開始做。
    • 先確認 $'A'$ 是一個原子語句，故 $'A'$ 為一完構式。
    • 再確認 $'(B↔C→D)'$ 不是一個原子語句，故前進至步驟 2。
    • 判斷該表達式沒有主要連接詞，故整個表達式都不是完構式。
從以上對此例的分析可發現，儘管我們對 $'(B↔C→D)'$ 的分析尚未全部完成，但既然已經確認到這一部分的表達式沒有主要連接詞之後，就不用再繼續分析下去，也能立即知道這整個表達式 $'(A∧(B↔C→D))'$ 都不是完構式。
