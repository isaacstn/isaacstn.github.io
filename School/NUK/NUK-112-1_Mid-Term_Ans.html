一、是非題 （每題 2 分）
閱讀以下敘述，正確請填 'T'，錯誤請填 'F'，不知道請填 'N'。

  1. 當一個論證有效，且前提全部為真時，則結論一定是真的。
    T：這就是有效論證的定義。
  2. 當一個論證有效，且前提當中有假時，則結論一定是假的。
    F：根據有效論證的定義，只能在前提全部為真時斷定結論一定也為真；故無法在前提有假時斷定結論一定也為假。
  3. 當一個論證無效，且前提全部為真時，則結論一定是假的。
    F：無效論證的定義只有要求在論證的前提皆為真時結論仍可能為假，並未要求在前提皆為真時結論一定為假。
  4. 當一個論證無效，且前提當中有假時，則結論的真假不能判斷。
    T：無效論證的定義只有提到當論證前提皆為真時結論仍可能為真，並未提到當論證前提有假時結論是真還是假，故確實不能判斷結論的真假。
  5. 當論證的前提全部為真，但結論卻為假時，則該論證一定是無效的。
    T：根據定義，只要一個論證在前提皆為真的情況下結論仍有可能為假，此即為無效論證。如今既已確定一論證之前提皆為真但結論卻為假，自然就證明了那是一個無效論證。
  6. 當論證的前提全部為真，且結論也為真時，則該論證一定是有效的。
    F：當論證的前提和結論皆為真時，有兩種可能的情況：一、結論是因為前提皆真而為真，故只要前提皆真則結論必為真；二、結論和前提共同為真只是巧合，兩者間沒有關聯。只有在第一種情況下論證是有效的。但題目只說論證的前提和結論皆為真，卻並未指出是上述兩種情況中的哪一種，故不能判斷論證是有效的。
  7. 當一個論證有效，且結論為真時，則前提一定全部為真。
    F：根據有效論證的定義，只能從前提皆為真推論出結論為真，卻不能從結論為真推論出前提皆為真。
  8. 當一個論證有效，且結論為假時，則前提當中一定有假。
    T：根據有效論證的定義，只要前提皆為真，則結論一定為真，現既已確認結論為假，那前提就不可能全部為真，否則結論必真。故前提當中一定有假。
  9. 當一個論證無效，且結論為真時，則前提的真假不能判斷。
    T：根據定義，當論證無效時，只能知道在前提皆為真的情況下結論可能為假，因此無法從結論的真假推論出前提的真假。
  10. 當一個論證無效，且結論為假時，則前提一定全部為真。
    F：道理同上，當論證無效時，無法從結論的真假推論出前提的真假。
  11. 在二值的語言中，所有語句都不能既真又假，但有些語句可以不真不假。
    F：既然是二值的語言，那麼所有語句都不能既真又假，也不能不真不假。
  12. 語言的組合性讓我們可以分析語句的句型和結構。
    T：正因為語言有組合性，我們才能將不同的語句以特定的方式結合，因此語句才會有複雜的結構，我們也才能從語句的複雜結構中區分出各種句型。
  13. 語句邏輯中的一元連接詞不可以連接超過一個語句。
    T：一元連接詞必須連接剛好一個語句，故確實不可以連接超過一個語句。
  14. 語句邏輯中的二元連接詞可以連接少於二個語句。
    F：二元連接詞必須要連接兩個語句，故不能連接少於兩個語句。
  15. 如果所有人都不知道一個語句的真假，該語句就沒有真假可言。
    F：語句的真假是根據語句表達的內容與事實是否相符來決定，只要語句有表達內容，無論如何都會有相對應的事實，只要語句表達的內容符合事實即為真、不符合事實即為假，因此無論人們知不知道那個事實是如何，都不影響語句的真假。
  16. $(¬A)∧B$ 也可以被寫成 $¬A∧B$。
    T：括號內的連接詞 '$¬$' 比括號外的連接詞 '$∧$' 更優先，故可以省略。
  17. $H→(I∧J)$ 也可以被寫成 $H→I∧J$。
    T：括號內的連接詞 '$∧$' 比括號外的連接詞 '$→$' 更優先，故可以省略。
  18. $¬(C↔D)$ 也可以被寫成 $¬C↔D$。
    F：括號內的連接詞 '$↔$' 不比括號外的連接詞 '$¬$' 更優先，故不可以省略。
  19. $(E∨F)∧G$ 也可以被寫成 $E∨F∧G$。
    F：括號內的連接詞 '$∨$' 不比括號外的連接詞 '$∧$' 更優先，故不可以省略。
  20. $(L∧M)→O)↔(¬M∨O)$ 不是一個完構式。
    T：左括號比右括號少了一個，故不是完構式。
  21. $¬S→T→U$ 不是一個完構式。
    T：在所有括號的最外層共有兩個 '$→$' 連接詞，它們的優先順序相同，故無法確定表達式的主要連接詞是哪個，因此不是完構式。
  22. $K∧¬K$ 不是一個完構式。
    F：由於 $K$ 是語句符號，故是完構式，因此將$K$代入規則(2)所得的$¬K$也是完構式。由於$K$和$¬K$都是完構式，將二者代入規則(3)所得的$K∧¬K$當然也是完構式。
  23. $((¬A)∨(¬Z))↔(¬(A∧Z))$ 是一個雙條件句。
    T：唯一不在任何括號中的連接詞就是 '$↔$'，故該語句確為雙條件句。
  24. $¬((V∨¬V)∧(W→W))$ 是一個連言句。
    F：唯一不在任何括號中的連接詞應是位在最左邊的 '$¬$'，故該語句實為否定句。
  25. $¬(B∨C)↔(D→E)$ 是一個否定句。
    F：語句中不在任何括號內的連接詞有兩個，分別是位在最左邊的 '$¬$' 和位在中間的 '$↔$'，由於後者的優先順序比前者更低，所以後者才是該語句的主要連接詞，因此該語句應為雙條件句。

二、簡答題 判斷蘊涵關係是否成立（作答方式同是非題）。（每題 2 分）
先說明蘊涵的關係。$A$蘊涵$B$即意味著只要$A$成立$B$就一定成立；因此，一但確認了只要$A$成立$B$就一定成立的關係，即能斷定$A$蘊涵$B$。

  1. 「是單身漢」蘊涵「是男人」
    T：只要是單身漢就一定是男人，故蘊涵關係成立。
  2. 「內角和180度的多邊形」蘊涵「三角形」
    T：只要是內角和180度的多邊形就一定是三角形，故蘊涵關係成立。
  3. 「是白馬」蘊涵「是馬」
    T：只要是白馬就一定是馬，故蘊涵關係成立。
  4. 「是馬」蘊涵「是白馬」
    F：是馬不一定就是白馬，也可能是紅、黃、黑馬，故蘊涵關係不成立。
  1. 「是生物」蘊涵「是動物」
    F：是生物不一定就是動物，也可能是植物，故蘊涵關係不成立。
  2. 「是動物」蘊涵「是生物」
    T：只要是動物就一定是生物，故蘊涵關係成立。
  3. 「能被2整除的數」蘊涵「偶數」
    T：只要是能被$2$整除的數就一定是偶數，故蘊涵關係成立。
  4. 「有顏色」蘊涵「是紅色」
    F：有顏色不一定就是紅色，也可能是黃、藍、綠色，故蘊涵關係不成立。

四、簡答題 判斷論證是否有效，並說明理由。（每題 5 分）

  1. 是人就會犯錯。
  2. 黃頌竹是人。
  3. 黃頌竹會犯錯。
    4. 答：有效。因為當論證前提全為真時，表示所有人都會犯錯，既然黃頌竹也是人，他當然也會犯錯。
  1. 塑膠不會導電
  2. 教室裡的花瓶不會導電。
  3. 教室裡的花瓶是塑膠做的。
    4. 答：無效。教室裡的花瓶有可能是用塑膠外的不導電材質做的，故當論證前提全真時，結論仍可能假。
  1. 犯人是司機或園丁。
  2. 犯人不是司機。
  3. 犯人是園丁。

    4. 答：有效。因為當論證前提全為真時，表示犯人只能是園丁或司機，但司機不是犯人，因此只能是園丁。

  1. $(x+y)^2=16$
  2. $x=4$
  3. $y=0$

    4. 答：無效。因為當論證前提全為真時，$(4+y)^2=16$，則 $4+y=±4$，因此有可能 $y=0$，但也可能 $y=-8$。

  1. 一年最多只有三百六十六天。
  2. 全校有超過三百六十六個同學。
  3. 至少有兩個同學在同月同日出生。

    4. 答：有效。一年最多只有366天，而全校卻有超過366位同學；假如有366位同學剛好生日都不相同，那麼一年內每天都有人生日，因此從第367人起就會和前面的366位同學中的某人同月同日生，假如那366位同學並非生日都不相同，那麼這366位同學中至少就會有2人同月同日生。

  1. 鴨嘴獸是哺乳類。
  2. 大部分的哺乳類都是胎生動物。
  3. 鴨嘴獸是胎生動物。

    4. 答：無效。儘管根據前提2可知大部分的哺乳類動物都是胎生，但仍有少部分哺乳類動物是否屬於胎生動物是未知的，因此就算根據前提1可知鴨嘴獸為哺乳類動物，也未必就是胎生動物。且事實上已知鴨嘴獸為卵生的哺乳類動物，更證明了此論證在前提全為真時結論仍可以為假。

  1. 如果汽車沒有燃料就不能發動。
  2. 汽車不能發動。
  3. 汽車沒有燃料。

    4. 答：無效。確實汽車沒有燃料就不能發動，但若汽車有燃料時卻沒有電，那同樣也不能發動，故不能確定汽車不能發動一定是因為沒有燃料。

  1. 只要颱風過境，村莊就會淹水。
  2. 村莊一整年都沒有淹水。
  3. 颱風一整年都沒有過境。

    4. 答：有效。既然只要有颱風過境村莊就一定淹水，那麼當村莊一整年都未淹水時，就表示颱風一整年都未過境，否則這一年內就一定淹過水。
