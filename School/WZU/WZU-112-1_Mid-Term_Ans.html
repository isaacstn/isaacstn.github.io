一、是非題 （每題 2 分）
閱讀以下敘述，正確請填 'T'，錯誤請填 'F'，不知道請填 'N'。

  1. 對一個有效論證來說，當前提全部為真時，論證的結論就是真的。
    T：這就是有效論證的定義。
  2. 如果論證的前提全部為真，但結論卻是假的，則該論證一定是無效的。
    T：只要在前提皆真的情況下，結論仍可能為假，論證就是無效的。因此在已經確定前提皆真而結論為假時，當然可斷定論證無效。
  3. 對一個有效論證來說，論證的結論一定是真的。
    F：有效論證只有在確認前提皆真的情況下可以保證結論為真。因此若不先確認前提的真假便無法確認結論是否為真。
  4. 對一個無效論證來說，論證的結論一定是假的。
    F：無效論證只是結論在前提皆真的情況下仍可能為假，並不是說只要是無效論證的結論就一定為假。
  5. 對一個無效論證來說，當前提全部為真時，論證的結論仍有可能是真的。
    T：無效論證的結論在前提皆真的情況下只是可能為假，因此當然不排除它也可能為真的情況。
  6. 如果論證的前提全部為真，且結論也是真的，則該論證一定是有效的。
    F：當論證的前提和結論皆為真時，有兩種可能的情況：一、結論是因為前提皆真而為真，故只要前提皆真則結論必為真；二、結論和前提共同為真只是巧合，兩者間沒有關聯。只有在第一種情況下論證是有效的。但題目只說論證的前提和結論皆為真，卻並未指出是上述兩種情況中的哪一種，故不能判斷論證是有效的。
  7. 在二值的語言中，所有語句都不能既真又假，但有些語句可以不真不假。
    F：既然是二值的語言，那麼所有語句都不能既真又假，也不能不真不假。
  8. 即使所有人都不知道一個語句的真假，該語句仍然有真假可言。
    T：確實。語句的真假是根據語句表達的內容與事實是否相符來決定，只要語句有表達內容，無論如何都會有相對應的事實可以比對，只要語句表達的內容符合事實即為真、不符合事實即為假，因此無論人們知不知道那個事實是如何，都不影響語句的真假。
  9. 語言的組合性讓我們可以分析語句的句型和結構。
    T：正因為語言有組合性，我們才能將不同的語句以特定的方式結合，因此語句才會有複雜的結構，我們也才能從語句的複雜結構中區分出各種句型。
  10. 在自然語言中，語句的意義結構都有被文字結構完整地呈現。
    F：邏輯學家之所以要創造出邏輯語言，其中一個主要理由就是要改變自然語言的語句結構常常會因為文字表面結構的多變性給隱藏或誤導，因此語句的意義結構當然沒有被文字結構完整呈現。
  11. 語句邏輯中的二元連接詞可以連接少於二個語句。
    F：二元連接詞必須要連接兩個語句，故不能連接少於兩個語句。
  12. 語句邏輯中的一元連接詞不可以連接超過一個語句。
    T：一元連接詞必須剛好連接一個語句，故不可以連接超過一個語句。
  13. 簡單語句的意義結構中不能包含任何語句連接詞。
    T：任何語句的語意結構中若包含了任何連接詞，就是一個複雜的語句，因此簡單語句當然不能包含任何連接詞。
  14. 只要一個語句的意義結構中包含任何語句連接詞就算是複雜語句。
    T：任何語句的意義結構中若包含了任何語句連接詞，就是一個複雜的語句。
  15. 複雜語句一定是由多個原子語句組成的，所以「科科不會邏輯」不是複雜語句。
    F：任何語句的意義結構中只要包含了任何語句連接詞就算是複雜的語句了，既然語句連接詞中有一元連接詞，而一元連接詞又只能連接一個語句，因此有些複雜的語句只會由一個原子語句所構成，不一定都是由多個原子語句構成。例如「科科不會邏輯」就是用「科科會邏輯」這個原子語句加上「並非」這個一元連接詞所構成的複雜語句。

二、選擇題 根據語義，判斷語句為何種句型。（每題 3 分）

  1. 科科和安安都是大學生。
    (A)原子語句；(B)連言句；(C)選言句
    B：本句應理解為「科科是大學生，而且，安安是大學生」，主要連接詞是「而且」，故為連言句。
  2. 科科讀哲學系但安安不是。
    (A)連言句；(B)否定句；(C)條件句
    A：本句當中的「但是」之意思與「而且」相同，故本句應直接理解為「科科讀哲學系，而且，並非安安讀哲學系」，主要連接詞仍然是「而且」，故為連言句。
  3. 安安讀哲學系或數學系。
    (A)連言句；(B)選言句；(C)雙條件句
    B：本句應理解為「安安讀哲學系，或者，安安讀數學系」，主要連接詞為「或者」，故為選言句。雖然本句亦可理解為「如果安安不是讀哲學系，則安安讀數學系」，那就會是條件句，但選項中並無條件句，因此還是只能選選言句。
  4. 科科和安安是同班同學。
    (A)連言句；(B)原子語句；(C)條件句
    B：「是同班同學」至少需要兩個東西作為主詞，故本句無法被拆解，故為一原子語句。
  5. 科科和安安都不是研究生。
    (A)連言句；(B)否定句；(C)雙條件句
    A or B：本句應理解為「並非科科是研究生，而且，並非安安是研究生」，主要連接詞為「而且」，故為連言句。不過本句亦可理解為「並非：科科或安安是研究生」，如此則主要連接詞會是「並非」，因此會是否定句，故選否定句也算對。
  1. 科科和安安並不是都讀哲學系。
    (A)連言句；(B)雙條件句；(C)否定句
    C：本句應理解為「並非：科科和安安都讀哲學系」，雖然其中的「科科和安安都讀哲學系」如題1一樣是連言句，但因為這個連言句又被最前面的「並非」否定了，所以「並非」才是本句的主要連接詞，故為否定句。誠然，此句亦可被理解為「並非科科讀哲學系，或者，並非安安讀哲學系」，如此一來本句便是選言句，但由於選項中並無選言句可選，因此還是只能選否定句。
  2. 科科或安安修過邏輯。
    (A)選言句；(B)連言句；(C)原子語句
    A：本句應理解為「科科修過邏輯，或者，安安修過邏輯」，主要連接詞為「或者」，故為選言句。當然本句亦可理解為「如果不是科科修過邏輯那就是安安修過邏輯」，如此則本句應為條件句，但由於選項中並無條件句，因此仍只能選選言句。
  3. 安安沒修過邏輯。
    (A)原子語句；(B)連言句；(C)否定句
    C：本句應理解為「並非安安修過邏輯」，主要連接詞為「並非」，故為否定句。
  4. 只要科科出席就會過。
    (A)雙條件句；(B)選言句；(C)條件句
    C：本句應理解為「如果科科出席，則科科會過」，主要連接詞為「如果……則……」，故為條件句。
  5. $n$ 是偶數若且唯若 $2$ 是其因數。
    (A)雙條件句；(B)選言句；(C)連言句
    A：本句的主要連接詞很明顯是「若且唯若」，故為雙條件句。

三、簡答題 （每題 3 分）
判斷下列組合是充分條件、必要條件還是充要條件的關係。
先分別說明充分條件、必要條件，和充要條件這三種關係：
A是B的充分條件：只要A成立則B也會成立；
A是B的必要條件：只要A不成立則B也不會成立；
A是B的充要條件：只要A成立則B也成立，且只要A不成立則B也不會成立。

  1. 「三角形」之於「有三個邊的多邊形」
    充要條件，因為三角形一定是有三個邊的多邊形；而且只要不是三角形就一定不是有三個邊的多邊形。
  2. 「有三個邊的多邊形」之於「三角形」
    充要條件，因為只要是有三個邊的多邊形就一定是三角形；而且只要不是有三個邊的多邊形就一定不是三角形。
  3. 「是白馬」之於「是馬」
    充分條件，因為白馬一定都是馬，但就算不是白馬也可能是馬。
  4. 「是馬」之於「是白馬」
    必要條件，因為不是馬就一定不是白馬，但馬卻不一定都會是白馬。
  5. 「是生物」之於「是動物」
    必要條件，因為不是生物就一定不是動物，但生物卻不一定都會是動物。
  1. 「是動物」之於「是生物」
    充分條件，因為動物一定是生物，但就算不是動物也可能是生物。
  2. 「有顏色」之於「是紅色」
    必要條件，因為沒有顏色就不會是紅色，但就算有顏色也不一定是紅色。
  3. 「是單身漢」之於「是男人」
    充分條件，因為單身漢一定都是男人，但不是單身漢也可以是男人。
  4. 「能被 $2$ 整除的數」之於「偶數」
    充要條件，因為能被 $2$ 整除的數一定都是偶數，而且不能被 $2$ 整除的數都一定不是偶數。
  5. 「偶數」之於「能被 $2$ 整除的數」
    充要條件，因為偶數一定都能被 $2$ 整除，而且不是偶數的數一定都不能被 $2$ 整除。

四、問答題 （每題 5 分）
判斷以下論證是否有效，並說明理由。

  1. 是人就會犯錯。
  2. 黃頌竹是人。
  3. 黃頌竹會犯錯。
    4. 答：有效。因為當論證前提全為真時，表示所有人都會犯錯，既然黃頌竹也是人，他當然也會犯錯。
  1. 塑膠不會導電
  2. 教室裡的花瓶不會導電。
  3. 教室裡的花瓶是塑膠做的。
    4. 答：無效。教室裡的花瓶有可能是用塑膠外的不導電材質做的，故當論證前提全真時，結論仍可能假。
  1. 司機或園丁是犯人。
  2. 司機不是犯人。
  3. 園丁是犯人。

    4. 答：有效。因為當論證前提全為真時，表示犯人只能是園丁或司機，但司機不是犯人，因此只能是園丁。

  1. $(x+y)^2=16$
  2. $x=4$
  3. $y=0$

    4. 答：無效。因為當論證前提全為真時，$(4+y)^2=16$，則 $4+y=±4$，因此有可能 $y=0$，但也可能 $y=-8$。

  1. 一年共有十二個月。
  2. 班上有超過十二個同學。
  3. 至少有兩個同學在同月出生。

    4. 答：有效。一年只有12個月，而班上卻有超過12位同學；假如有12位同學都在不同月份出生，那麼一年內每個月就都有人出生，因此從第13人起就會和前面的12位同學中的某人同月出生，假如那12位同學並非都在不同月份出生，那麼這12位同學內至少就會有2人在同月出生。

  1. 鴨嘴獸是哺乳類。
  2. 大部分的哺乳類都是胎生動物。
  3. 鴨嘴獸是胎生動物。

    4. 答：無效。儘管根據前提2可知大部分的哺乳類動物都是胎生，但仍有少部分哺乳類動物是未知的，因此就算根據前提1可知鴨嘴獸為哺乳類動物，也未必就是胎生動物。且事實上已知鴨嘴獸為卵生的哺乳類動物，更證明了此論證在前提全為真時結論仍可以為假。

  1. 汽車如果沒有燃料就不能發動。
  2. 汽車不能發動。
  3. 汽車沒有燃料。

    4. 答：無效。確實汽車沒有燃料就不能發動，但若汽車有燃料時卻沒有電，那同樣也不能發動，故不能確定汽車不能發動一定是因為沒有燃料。

  1. 只要颱風過境，村莊就會淹水。
  2. 村莊一整年都沒有淹水。
  3. 颱風一整年都沒有過境。

    4. 答：有效。既然只要有颱風過境村莊就一定淹水，那麼當村莊一整年都未淹水時，就表示颱風一整年都未過境，否則這一年內就一定淹過水。
